O hře (top-down zombie shooter)

Hráč vidí mapu shora (bird view), pohybuje se po učebně, střílí zombie a snaží se přežít co nejdéle.

Pravidla hry

  • WASD/šipky = pohyb hráče.
  • Levé tlačítko myši nebo mezerník = střelba.
  • Zombie se spawnují na okrajích mapy a jdou po hráči.
  • Po zásahu hráč ztrácí HP; při 0 HP končí hra.

Funkční požadavky

  • Herní smyčka (update + render).
  • Menu se Start/Konec + game over obrazovka.
  • Skóre podle počtu zabitých zombie + doby přežití.
  • Ukládání výsledků do souboru i databáze.

Technické požadavky

  • Python + Pygame pro samotnou hru.
  • SQLite pro ukládání výsledků.
  • Webová prezentace + jednoduché webové DB rozhraní (registrace, žebříček).
  • Modulární kód, komentované funkce/třídy a verzování v Gitu.
